# Python Flask Web Development Projects

This repository contains several web development projects built using **Python, Flask, and HTML5**.
The projects were developed to practice backend web development, database integration, and building simple web applications.

## Technologies Used

* Python
* Flask
* HTML5
* CSS
* SQLite / Database connection

## Projects

### 1. Event Registration System

A simple web application that allows users to register for an event through a web form.

**Features**

* User registration form
* Collect participant information
* Store data into database

**Technologies**

* Flask
* HTML5
* Python

---

### 2. Student Management System

A basic system that allows users to manage student information with database integration.

**Features**

* Add student information
* View student list
* Store and retrieve data from database

**Technologies**

* Python
* Flask
* SQLite / Database

---

### 3. Flask Student Management Web App

A web-based student management application developed using Flask framework.

**Features**

* Manage student records
* Display data on web interface
* Basic CRUD operations

**Technologies**

* Flask
* Python
* HTML5

---

## Project Purpose

These projects were developed as part of academic practice to improve skills in **Python web development and Flask framework**.
